## 📥 Pasos para descargar e instalar el Agente de Impresión

Sigue estos pasos para obtener y ejecutar el agente en tu equipo:

1.  **Haz clic en el enlace de arriba**  
    Dirígete al repositorio de GitHub del Agente de Impresión.

2.  **Haz clic en el botón verde `<> Code`**  
    Se encuentra en la parte superior derecha de la página del repositorio.

3.  **Selecciona `Download ZIP`**  
    Se desplegará un menú, elige esta opción para descargar el archivo comprimido.

4.  **Descomprime el archivo ZIP**  
    Extrae el contenido en la ubicación que prefieras en tu computadora.

5.  **Abre la carpeta `AgenteImpresion`**  
    Dentro de los archivos extraídos, busca y abre esta carpeta.

6.  **Sigue las instrucciones internas del CMD**  
    Ejecuta el archivo correspondiente y sigue los pasos que se muestran en la ventana de comandos (CMD) para completar la instalación y configuración.

> ⚠️ **Nota:** Asegúrate de tener los permisos necesarios y una conexión estable a internet durante el proceso.
